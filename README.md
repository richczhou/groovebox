### Install

---

1. Clone the repository:

```bash
git clone https://github.com/richczhou/groovebox.git
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server and open [http://localhost:3000](http://localhost:3000):

```bash
npm run dev
```
